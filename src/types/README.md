# TypeScript Types

Dieses Verzeichnis enthält TypeScript-Typdefinitionen für die gesamte Anwendung.

## Contents
- Schnittstellen für Chat-Nachrichten und Konversationen
- Typen für API-Anfragen und -Antworten
